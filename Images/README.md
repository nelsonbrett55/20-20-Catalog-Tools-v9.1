# Images

Screenshots and reference images used throughout the **[20-20 Catalog Tools v9.1](../)** guides.

All images live loose in this folder. Each guide in the main repo links to the specific files it needs, so you don't normally need to browse here directly — but this is where every screenshot, diagram, and reference picture is stored.

### A few notes
- **Editing a guide?** Add any new screenshots here and reference them from your Markdown with a relative path, e.g. `![Variable tab](../Images/variable-tab.png)`.
- **Naming:** Try to give files clear, descriptive names (like `hierarchy-add-item.png`) so they're easy to find and reuse.
- **Formats:** PNG is preferred for screenshots; JPG is fine for photos.
