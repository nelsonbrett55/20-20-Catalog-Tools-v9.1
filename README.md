# 20-20 Catalog Tools v9.1 - Guide for Catalog Editors

A practical, hands-on guide to using **2020 Catalog Tools v9.1** (Cyncly / 2020 Design) for building, editing, and maintaining manufacturer catalogs. Written by a catalog editor, for catalog editors - covering everything from the basics of each tab to advanced tricks like expression-based data, automatic addons, and catalog cloning.

> **Who is this for?** Kitchen & bath dealers, manufacturers, and catalog editors who work with 2020 Design catalogs and want to go beyond the official documentation.
